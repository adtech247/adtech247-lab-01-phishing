# 🛠️ Setup / Getting Started

Good news: **this lab requires zero setup.** No software to install, no accounts to create.

## How to Do This Lab

**Option 1 — Right here on GitHub (easiest)**
Just read the files in your browser:
1. Read [`README.md`](README.md) for the intro and checklist
2. Open [`challenge/emails.md`](challenge/emails.md) and work through the 5 emails
3. Check your answers in [`solution.md`](solution.md)

**Option 2 — Clone it locally**
If you'd rather work offline or take notes in the files:
```bash
git clone https://github.com/YOUR-USERNAME/adtech247-lab-01-phishing.git
cd adtech247-lab-01-phishing
```
Then open the files in any text editor or markdown viewer.

## What You Need

- 15 minutes
- Your eyes
- A willingness to be tricked (so you learn not to be) 🙂

That's it. Start with the [`README.md`](README.md)!
